# Python企业员工管理系统简介

## 核心功能
- 登录注册
- 员工信息总览
- 打卡考勤数据可视化
- 员工打卡
- 员工考勤
- 员工请假
- 员工薪资
- 部门管理
- 员工管理
- 请假申请管理
- 个人中心

## 适用场景
- 企业内部员工管理
- 考勤统计
- 薪资管理
- 部门组织架构管理

## 技术特点
- 采用Django框架进行后端开发
- 使用MySQL数据库进行数据存储
- 前端采用Vue.js进行界面展示
- 支持员工和管理员两种角色访问
